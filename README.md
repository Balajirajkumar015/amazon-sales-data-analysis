# Amazon Sales Data Analysis

## Objective
To analyze Amazon sales data and identify revenue trends, order status patterns,
top-performing states, and fulfilment performance.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Jupyter Notebook

## Dataset
Amazon sales dataset containing 120k+ records.

## Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Order status and cancellation analysis
- State-wise and monthly sales trends
- Fulfilment performance comparison

## Key Insights
- Amazon fulfilment contributes higher revenue
- Few states generate majority of sales
- Seasonal patterns affect monthly revenue

## Conclusion
This project provides actionable insights to support business decision-making.
